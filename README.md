# Sistema de Gerenciamento de Produtos | Python + SQLite3

Projeto desenvolvido com o objetivo de praticar operações CRUD (Create, Read, Update, Delete), manipulação de banco de dados e organização de código em funções reutilizáveis.

## Funcionalidades

- Operações CRUD completas
- Validação de dados de entrada
- Menu interativo via terminal
- Estruturação modular em funções

## Como Executar

1. Clone o repositório ou baixe os arquivos.
2. Execute o arquivo `products.py`:
3. O Banco de dados será criado na primeira execução.
